# EAcademy 🎓

EAcademy is a modern online learning platform developed as a Final Year Project (FYP).  
The system helps students and teachers manage courses, assignments, quizzes, study material, and communication in one platform.

---

# 📌 Features

- Student Authentication
- Teacher & Admin Dashboard
- Course Management
- Assignment Upload & Submission
- Online Quiz System
- Notes & Study Material Sharing
- Student Performance Tracking
- Notifications & Announcements
- Responsive User Interface

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/EAcademy.git
```

## Move to Project Folder

```bash
cd EAcademy
```

## Install Frontend Dependencies

```bash
cd client
npm install
```

## Install Backend Dependencies

```bash
cd server
npm install
```

---

# ▶️ Run Project

## Run Backend

```bash
cd server
npm start
```

## Run Frontend

```bash
cd client
npm start
```
